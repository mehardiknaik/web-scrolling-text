## Running Examples Locally

Clone the repository and navigate to any example:

```bash
git clone https://github.com/mehardiknaik/web-scrolling-text.git
cd web-scrolling-text/examples/react  # or nextjs, angular, vanilla
npm install
npm run dev
```
